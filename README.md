# Ogilvy C2 Career Goal Framework

Interactive prototype for Ogilvy C2 employees to browse 2026 SMART career goal examples by discipline and theme, then adapt selected goals into a personal 2026 plan.

## Files

- `index.html` - prototype page structure
- `styles.css` - visual system and responsive layout
- `app.js` - goal data, filtering, planner interactions, and scroll motion

Open `index.html` in a browser to run the prototype.
